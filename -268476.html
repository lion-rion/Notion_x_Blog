書いた理由 ・私の通っている立命館大学情報理工学部は大学の中では難関と呼ばれる部類なので，入ってくるのは「頭はいいけど情報系の知識はほぼ0」という子が多いです．僕は頭も悪く(追加合格者)，情報系の知識もほぼ0だったのですが，どのようにして今に至ったのかを書きたいと思います．

まずは自己紹介

石田優希 (ハンドルネーム : Lion)

  • 立命館大学 情報理工学部 情報理工学科 セキュリティ・ネットワークコース 2回生
  • 言語 : C Python PHP JavaScript
  • 得意分野 : Web

全体ロードマップ

1年生のときは基本HTMLとCSSしかやってません．僕のGithubをみたらわかりますが，技術が付き始めたのは2年生からなので今焦っている子は全然焦らなくてもいいと思います．

ここからは実際に振り返りつつ色々書いていきます．

受験終了後 2月~6月

Web制作に挑戦

とりあえず毎日ゲームをしつつ，高校生の頃からやってみたかったWebデザイン・Web制作に手を出し，Progateを始める．

Progateを終わらせた後何をしたらいいのかわからず質問サイトに何をしたらいいのかを質問しまくりました．

  • この時みんな口を揃えて「作りたいものを作ってみましょう」と解答してました

作りたいものがなかったので，おすすめされた模写コーディングというものをやってみた．

📌

模写コーディングとはそのままの意味で，Webサイトを見たりツールを使ったりしながら同じサイトを作ってみることです．私は「蒼乃建設の模写」というものをやりました！

https://vinaka-design.net/copy/dan/

今では超簡単ですが，初心者にはかなり難しいので結局あきらめ，当時ドハマリしていたRustというPCゲームの攻略サイトを作ってみたくなり作りはじめました．

知識も全くなかったので質問サイトに質問してはボコボコに馬鹿にされたり，デザインをうまく反映できずに困っていました．

特にリセットcssの存在を知らなかったのは痛かったですね…

Macでイキる

入学するころにはVscodeでちょっとかけるくらいになっていたのでMacでドヤ顔をしながらVscodeを開いて書いていました😎  正直イキることが一番のモチベでした…

知るカフェでドヤ顔コーディングしたり，大教室の前のほうでイキってカキカキしたり英語の授業中にも書いたりしてドヤってました．

SEOに興味を持つ

試行錯誤の上，ある程度形になってきたので実際にサイトを登録をしたり，SEOを考えたりするようにしました．

📌

SEOとは検索エンジンの結果において，上位表示を狙えるようにサイトを最適化したり，ユーザビリティを考えたサイト構成をすることです．

SEOの観点には以下のようなものがあります．

  • サイトの表示までの時間が早いか
  • スマホ向けに最適化されているか
  • タグを適材適所で使っているか
  • ページ数が多いか

ここで重大な問題がありました．

この時僕のサイトの比率は3:7でナビゲーションとボティでした．そのためスマホの画面に合わせて調節するのがすごく難しかったんです．そのためスマホ向けの最適化ですごく苦労をしていたのでサイトを一新することにしました．

夏休み入 8月~11月

サイトを一新することにしてから夏休みはひたすらHTMLとCSSを書いていました．このときもまだWebアプリやJavaScriptを知らなかったのでずっとHTMLとCSSだけで戦ってました．

ここから試したものは以下の通りです．

  • 画像圧縮 CSS圧縮
  • レスポンシブ対応
  • WordPress
  • 3カラム
  • 検索エンジンの仕組みを勉強
  • Google最強！！！！

こんな感じです．圧縮から何から何まで全部手作業でやってたので平気で時間が溶けました．今では絶対に考えられないですね… でもそのおかげでHTMLとCSSの理解や検索エンジンとお友達になれたのでOKです．

一人暮らし開始 12月~3月

僕は地元勢で，実家から大学に通っていたのですが，12月にいろいろあり親と喧嘩したので一人暮らしをすることになりました．

PHPを始める

一人暮らしを始めた影響か，何が理由かは覚えていませんが，PHPをはじめました．PHPと言ってもProgateをちょこっとやってLaravel入門のYouTubeを見ただけなのでPHPというよりLaravelの知識だけでPHPを語ってる状態です．

↓Laravel 5.xですが，すごくわかりやすかったのでおすすめです．

【Laravel入門・準備編】PHPフレームワークの種類と人気を紹介 #00

今回からLaravel入門ということで、PHPのフレームワークであるLaravelを学んでいきます！ 初回ということで、基本的なことを説明しますので、フレームワークについて全く知らないという方はぜひ参考にしてみてください！ 目次) ・そもそもフレームワークって？ ・PHPフレームワークの種類 ・PHPフレームワークの人気・求人数 ・Laravelで学ぶこと！ ...

https://youtu.be/cO9Kfh3lypg

また，ここあたりで作っていたWebアプリでJavaScriptが必要だったのでJavaScriptの知識もすこし付けました．ただ，Jsは必要なときに調べて使う感じにしているのでネットから引っ張ってきて使うくらいしかできません…

コマンドラインやSQLも強くなった

LaravelやPHPではcomposerというPHPパッケージ管理ツールやarisanコマンドをCUIで使うのでコマンドラインにも強くなりました．

またSQLも使うのでエラーについて調べるうちにそこそこ強くなり，現在(秋セメスタ)のデータベースの授業はある程度理解できてます．(なお中間テストボコボコ…)

2年生 4月から10月

イベントにいっぱい出た

4月からサークル関係もありWebセキュリティに手を出し始めました．というより義務でやってたので義務セキュリティです(笑)

徳丸本という有名な本を使って勉強していました．まだ4分の1も読めてません…😭

1回生に教えている時間が多かったですが，イベント関連にツヨツヨさんたちと出たことで超知識を付けられました．

出たイベント関連

  • オープンキャンパス・サマースクール
  • GMOインターネットグループ インフラインターン
  • GMOインターネットグループ セキュリティインターン
  • 第7回千葉大学バグハンティングコンテスト (優秀賞受賞)
  • JPHACKS2022
  • KC3(関西情報系団体交流会)

この中でもGMOのインフラインターンは僕がインフラ関連に興味を持つ大きなきっかけになりました．これがなかったらサーバーやネットワーク，CI/CD，Dockerあたりに手を出していなかったと思います．

普段やってたこと

イベントに出ることが多かったのですが，普段は何をしていたのかというと，基本雑用や事務業です．

というのも，2月に「たまたま1年生が僕しか残っていなかった」という理由でサークルの団体長になりました．

運営とか別にやらずにいても良かったんですが，性格上そういうのができないタイプなので何から何までほとんど1人でやりました．引き継ぎが0で調べることばかりで事務に聞きまくったりして進めていました．この4月~10月で社会に出てから体験する「嫌な仕事」をいろいろ体験しました．

現在は優秀な後輩が動いてくれるのでほとんどやることはなくなりました．最高です．

結局ここでいいたかったのは，イベントで得られる知識がエグいってことです．正直普段は特に何もしていませんでしたが，短期集中で取り組んだもので，沢山のことを学びました．この学びが今の僕を構成していると言っても過言ではありません．ですので，是非勇気のある方はガンガンイベントに参加して知識をつけるといいでしょう！

現在 11月~

現在は中途半端に手を出しては放置している状態です．来年のイベントなどに向けて準備を始めたいのですが，やってみたいことが多すぎて困っている状態です．その中でも現在はサーバー関連の知識を付けているところです．特にログ読みやシステムの異常などを調べる力がないのでサーバーを触って知識を付けているところです．

まとめ

上から読んで来たらわかると思いますが，僕はツヨツヨと違ってそこまでスキルも知識もありません．ツヨツヨに見えている一般学生で，まだまだまだまだヒヨコ🐥🐥🐥です．

「先輩がヒヨコなら僕らはなんなんですか！？」ってよく言われますが，多分うずら卵です．(は？)

ただ，一言いうとしたら「焦らなくていいけど，焦ってないのはやばい」ですかね．

今焦っている子はいい感じです．焦ってない人は…まあまあ…

院進しない人は早くて3年の秋から就活開始です．そう考えると時間が全くないのは言わずもがなです．

大企業と呼ばれる企業には中高のときからコンピュータをやっているツヨツヨや高専から編入したチートマンがエントリーしてくるので，その人たちと戦うには今のうちから準備が必要だと僕は思っています．

ただ，僕は就活目的ではなく，単純に面白いからコンピュータを勉強しています．面白いと思ってやっている人に，就活目的の人間は絶対勝てないので，僕的にはまずプログラミングやコンピュータというものを好きになって，楽しんでもらいたいですね！ 好きでやってれば勝手に良い企業に入れると思います．

と，ここまでツヨツヨでもなく，就活も経験してない人間が語りちらしたわけですが，結局は行動するかしないかが一番の分かれ目なのでまずは行動するように努力するといいと思います！では頑張ってくださいね！
